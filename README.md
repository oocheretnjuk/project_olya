Getting started:

1. clone this repo
2. cd path/to/
3. npm install git+https://git@github.com/gulpjs/gulp.git#4.0 gulpjs/gulp-cli -g  // Install the latest Gulp + Gulp CLI tools globally
4. npm install
6. run "gulp" command to start
